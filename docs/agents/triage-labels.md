# Triage Labels

Use the default triage vocabulary:

| Role | Label |
| --- | --- |
| Needs maintainer evaluation | `needs-triage` |
| Waiting for reporter information | `needs-info` |
| Ready for AFK agent implementation | `ready-for-agent` |
| Ready for human implementation | `ready-for-human` |
| Will not be implemented | `wontfix` |

Do not create duplicate labels with equivalent meanings.
